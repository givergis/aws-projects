# Project 1: Static Website Hosting on AWS S3

## Overview
This project demonstrates how to host a static website using Amazon S3.

## Implementation Steps
1. Created an S3 bucket with a globally unique name
2. Disabled block public access
3. Enabled static website hosting
4. Applied bucket policy for public read access
5. Uploaded website files

## Security Considerations
- Public access limited to object read-only
- No write or delete permissions exposed

## Outcome
Successfully hosted a live static website on AWS using S3.
